About This Project

Created a game and a game controller with detachable magnetic components. In the game, you fight a boss and survive by using the detachable components, which are shields that each have different abilities.

  • Designed and built a physical game controller with detachable magnetic shield components using an ESP32, sensors, RFID, and custom 3D-printed parts.
  • Developed a Unity boss-fight game featuring a modular shield ability system, integrating real-world controller input through wireless communication.
  • Iterated through hardware prototyping, soldering, and enclosure design while debugging sensor data, power issues, and gameplay integration.
  • Developed everything myself.

Technologies Used

Unity ESP32 Embedding magnets into 3D printing Web sockets RFID readers
