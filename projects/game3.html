About This Project

Puzzle game where you use a visual scripting language to recreate fractals with the goal of helping students visually understand recursion.

  • Created a visual scripting language that executes from left to right, designed to be easy to learn for users with little coding experience.
  • Replicated recursion by pushing and popping array lists from a stack in C#, with step-by-step execution to visualize how recursion works.
  • Used the visual scripting language to build multiple levels featuring fractals that gradually increased in difficulty.
  • Developed everything myself.
