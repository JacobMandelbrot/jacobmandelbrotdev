About This Project

2D rhythm platformer inspired by Pac-Man that placed in the top ten in high school FBLA national competition.

  • Built platforming, power-up, combo, and rhythm-based gameplay systems.
  • Integrated score bonuses for jumping on the beat, reinforcing music-driven mechanics.
  • Designed a simple, accessible gameplay loop that rewards skillful play rather than punishing mistakes.
  • Developed the project as an early learning experience, gaining core game design and Unity fundamentals.
  • Created all pixel art and original music for the game.

Technologies Used

Unity C# Pixel grid map Music rhythm in Unity
